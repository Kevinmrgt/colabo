# Anti-Gravity - Regles Specifiques

## Identifiant agent
**Nom** : Anti-Gravity

## Contexte d'utilisation
Anti-Gravity intervient principalement pour :
- Les taches d'implementation specialisees.
- Les operations complexes necessitant un agent dedie.

## Regles specifiques

1. **Scope limite** : N'intervenir que sur les taches explicitement assignees dans le plan.
2. **Rapport detaille** : Fournir un rapport complet apres chaque intervention.
3. **Pas de modification silencieuse** : Toute modification doit etre visible et documentee.
4. **Tracabilite** : Toute modification effectuee via Anti-Gravity doit etre documentee dans `code_changes_log/`.
